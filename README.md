# Venkatesh Cotton Corporation Website

## Project Structure

```
project/
│
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── images/
├── icons/
└── README.md
```

## How to Run

Open `index.html` in your browser (preferably with Live Server) to view the site.

## Notes

- All CSS is in `css/style.css`.
- All JavaScript is in `js/script.js`.
- Tailwind is loaded via CDN.
- No frameworks or libraries added.
- All features and appearance are preserved.
